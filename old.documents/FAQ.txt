Q.mokoを使用しても大丈夫?
　A.mokoは基本的にUIを変更するスクリプトファイル
　　人に聞くよりも自分で判断する。自己責任が前提

===== 不具合全般 =====

Q.全般に動作が極端に遅い
　A.chromeを再起動させる。鯖が重いこともあるが、長時間使用していると仕様メモリが増大するので再起動してメモリを開放してみる

Q.バージョンアップしたが表示が乱れる
　A.chromeのキャッシュをクリアしてみる。
　　又、何かの不具合でエラーが出ている場合もあるので Chromeのツール → JavaScriptコンソールにエラーが出ていないか確認してみる
　　もし、エラーが出ていれば、どのような場合にこうするとこうなるのようにツールスレにエラー内容と合わせて報告すれば改善される見込み有り

Q.章/期が変わったらmokoがおかしくなった
　A.mokoは章/期を判別出来ない。鯖の統合なのでカードが増える等、データの不一致が発生するので当然。これは仕様なのでCookieごと削除してリフレッシュする。
　　Chromeの設定画面 → 詳細設定を表示... → コンテンツの設定 → すべてのCookieとサイトデータ...
　　から w3◯◯sengokuixa.jpを削除する。◯◯は該当するワールド番号(統合鯖の場合は一番小さい番号が該当する)
　　又、通期で仕様していてもカードのデータを記録(グループ順・基本兵種)し、合成や削除なのでそのカード無くなると記録したデータはそのままで蓄積されていく。
　　記録→合成・カード削除の頻度が高い場合は時折リセットすることが望ましい。兵士編成のグループ順の強制ソートのレスポンスに多少影響してくる。

Q.プルダウンメニューがでない
　A.mokoのバージョンアップによるオプションデータの不整合でエラーがでてる可能性大。
　　Chromeのツール → JavaScriptコンソール → Resources → LocalStorageのkey値ixa_moko_optionsをDelete。
　　または、mokoの設定 → 全般２ → localStorageの一部をクリアを試す。
　　Ctrl+Shift+JのJavaScriptコンソールでエラーが出てるいか確認できる。

Q.突然、mokoが起動しなくなった、どうしよう...
　A.何かのトラブルでlocalStorageのデータが破損したと思われてる
　　全て設定のやり直しになるがCookieを削除する
　　Chromeの設定画面 → 詳細設定を表示... → コンテンツの設定 → すべてのCookieとサイトデータ...
　　から w3◯◯sengokuixa.jpを削除する。◯◯は該当するワールド番号(統合鯖の場合は一番小さい番号が該当する)


====== 設定関連 ======

Q.moko設定はどこ？
　A.右上の発言の注意の下にある「Mokoの設定」

Q.敵襲音が鳴らない
　A.デフォルトではオフなのでmoko設定を確認。
　　加えて、統合敵襲警報かまたは他の敵襲表示がオンでないと鳴らない。

Q.敵襲音を変えたい
　A.音を変えるには、moko設定の敵襲音の欄にあるsrcに音源ファイルの直リンURLを入れるか、
　　Data URI Schemeを入れる。欄を空白にすればデフォルトの音が鳴る。

Q.mokoの設定データだけを削除したい
  A.mokoの設定データはChromeのlocalStorageにすべて保存されている
　　Chromeのツール → デベロッパーツールを選択するとウインドウが開く
　　Resources → LocalStorage → http://w3◯◯.sengokuixa.jp に格納されている
　　key:ixa_moko_optionsのvalueはmokoの設定データになる
　　valueを削除すれば設定データのみリセットされる


======= 機能関連 ======

Q.待機武将一覧の使い方
　A.右サイドバートップのIXA用ツール欄にリンクがある。
　　moko設定の選択肢「兵士編成ページから」は速いが討伐ゲージが取れない、「部隊編成ページから」は遅い。
　　TableSorter導入で、テーブルの項目欄をクリックするとソートされる。Shift+クリックで複数列選択可能。
　　武将のセットは一部隊ずつ。

Q.待機武将一覧を使うとソートがカードNo.昇順になる
　A.武将を探しやすくするための仕様。

Q.広域マップの使い方
　A.同盟情報ページに「同盟員の拠点を記憶」というボタンが追加されているのでそれを押すと拠点が記憶され、
　　広域マップに表示できるようになる。クリックした場所に相当する座標に飛べる。

Q.敵襲のデスクトップ通知の消し方
　A.Chromeの設定の深いところに「通知」というのがある。その例外管理からixaのurlを削除する。
　　または、通知ウィンドウのタイトルバーのスパナマークをクリック。

Q.マップの移動を履歴に記録って？
　A.ブラウザの戻るで戻れるということ。ページ遷移なしのAjax移動は履歴に残らないので、
　　戻るボタンを押しても戻れない。このオプションで戻れるようになる。
　　注意：座標を入力して移動する場合は両方(x・ｙ)の数値を入力しないと選択拠点に戻される

Q.合流に乗れない、ログアウトになる
　A.鯖が重い時はツールが弾かれる。オフにしてみよう。

Q.おすすめの設定
　A.デフォルト＋全般１の敵襲音、地図のマップ移動を履歴に記憶、内政の下位生産施設非表示、兵舎の小分け生産

Q.犬聖ﾌﾞﾙ
　A.武将は取引のページからランダムで取得。戦闘システムはixaとほぼ同じ。必要経験値も。スキル発動も。
　　ステ振りはランダムだが極振り傾向になるように重み付けしてある。兵法振りで育つと悲惨。
