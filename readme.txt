　　　移動時間の異なる部隊の着弾時間合わせ、時間差着弾など...
　　　その手間と待ち時間のイライラを解消する機能[出陣確認画面でのみ動作]
　　　■使用方法
　　　　1.任意の時間を指定
　　　　　　到着時間を入力し「この時間をセット」押下でセット完了。セットした到着時間に到着で自動出陣
　　　　　　(最初に入力されている時間は入力用のダミー。出陣確認中の部隊のリロード時の到着時間。「更新」で現在の到着時間に書き換わる)
　　　　2.出陣中の部隊の到着に合わせる
　　　　　　他の出陣中の部隊がある場合に出陣確認を開く、またはリロードすると、部隊名の上に出陣中部隊の各データテーブルが表示される。
　　　　　　指定したい部隊テーブルの「この時間をセット」でセット完了。指定部隊の到着時間後、到着するように自動出陣
	【仕様】
　　　　・"おまかせ出陣!にセットした到着時間"と"出陣確認中の到着時間"がマッチすると自動出陣するシンプルな仕様　
　　　　　（セットした到着時間 == 到着時間ではなく、その約1~2秒後が実際の到着時間になる。通信状況で多少変動することも有る)
	【注意】　　　
　　　　・セットしたウインドウ、タブはおまかせ出陣!の出陣完了まで開いた状態を保つ、また、リロードはしないこと
　　　　・加速部隊スキルがある部隊の場合は警告を出すようにしているが、この場合は出陣後に到着時間が変わるので、あらかじめ同距離で確認等の計測が必要
